🔍 Google Search Console

Organisk søgetrafik til ekkosoftware.dk fra Google

Forbind Google Search Console for at vise søgedata. Du skal bruge et Google OAuth 2.0 Client ID fra Google Cloud Console.

Sådan opretter du et Client ID →
  1. Gå til console.cloud.google.com og opret/vælg et projekt
  2. Aktivér Google Search Console API (APIs & Services → Library)
  3. Gå til APIs & Services → Credentials → Create Credentials → OAuth client ID
  4. Vælg Web application og tilføj dashboardets URL under "Authorised JavaScript origins"
  5. Kopiér Client ID og indsæt herover
